# Little Lemon Restaurant Website

A modern, responsive website for Little Lemon, a family-owned Mediterranean restaurant in Chicago. Features a homepage with hero, specials, about, testimonials, and a fully functional reservations form with validation and confirmation page.

## Features
- Responsive design (desktop + mobile)
- Home page with Hero, Weekly Specials, About Us, and Customer Testimonials
- Reservations form with date picker, available times, guest count, contact info, and client-side validation
- Confirmation page with booking summary
- Accessibility: semantic HTML, ARIA attributes, proper labels, focus states
- Unit tests for form validation and reducer logic
- Booking data persisted to localStorage

## Technologies
- React 18
- React Router v6
- react-datepicker
- Jest + React Testing Library
- CSS (custom, no frameworks)
